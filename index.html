Google Drive Cloud Sync

Access and sync your private ledger across all your devices using your private Google Drive storage.

How to configure credentials:
  1. Go to Google Cloud Console.
  2. Create a project and set up the OAuth Consent Screen.
  3. Add scopes: .../auth/drive.file.
  4. Create an OAuth Client ID for Web application.
  5. Add Authorized JS Origins: http://localhost:8080 https://mkmohitkalal.github.io
  6. Copy the Client ID and paste it above!
